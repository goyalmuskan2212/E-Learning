## Introduction - LearnRush.

LearnRush is a modern full-stack e-learning platform built using the MERN stack, designed to provide free access to courses and lectures through a clean and intuitive user experience. The platform supports role-based access control, allowing admins to manage courses and lectures, while learners can explore, study, and stream educational content seamlessly.

It features secure authentication, responsive UI, video-based learning, and a scalable architecture, making it suitable for real-world deployment and future expansion (such as paid courses or subscriptions).
